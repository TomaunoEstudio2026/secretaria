🔐 Acceso al sistema

Para ingresar, escribí la contraseña de acceso cuando el sistema lo solicite. El sistema se bloquea automáticamente por inactividad. La contraseña de eliminación (para borrar talleres o desbloquear tareas) es diferente y más segura.

📋 Notas y Tareas

La pestaña Notas permite registrar tareas pendientes del día. Cada tarea tiene un estado (pendiente / hecha) y puede bloquearse para protegerla de eliminaciones accidentales. Las notas se sincronizan en tiempo real con Firebase y están disponibles desde cualquier dispositivo.

🔧 Talleres

Registrá cada taller con nombre, docente, horario, días de clase y cupo máximo. Desde la lista podés ver los alumnos inscriptos, editar los datos del taller o eliminarlo (requiere contraseña si tiene alumnos). También podés generar imágenes de los horarios en dos formatos:

  • 🖥️ Imagen TV — horizontal 1280px, ideal para pantallas o WhatsApp
  • 📱 Historia IG — vertical 1080×1920px, lista para publicar en Instagram
👥 Alumnos

Buscá alumnos por nombre, DNI o taller. Podés ver todos los talleres en los que está inscripto cada alumno, editar sus datos o darlo de baja. La inscripción se realiza desde la ficha del taller correspondiente.

✅ Asistencia

Registrá la asistencia diaria por taller. El sistema guarda el historial completo. Podés ver el porcentaje de asistencia de cada alumno y exportar los datos si necesitás un informe.

📅 Calendario

Agendá eventos institucionales: actos, reuniones, feriados, capacitaciones. Cada evento tiene fecha, hora opcional y descripción. Los eventos marcados como 📺 Pantalla se muestran automáticamente en la Cartelería Digital sin necesidad de configurarlos dos veces.

📺 Cartelería Digital

Armá la lista de mensajes que se proyectan en la pantalla del centro. Cada renglón es un mensaje o slot especial. Podés:

  • Ordenar los mensajes arrastrando con ⠿
  • Activar o desactivar cada mensaje con el ✓
  • Configurar cuántos segundos se muestra cada uno
  • 📅 Eventos calendario — inserta automáticamente los próximos eventos tipo Pantalla
  • 🔧 Talleres x taller — una slide por cada taller con nombre, docente y horario
  • 📋 Placa talleres — tabla completa de todos los talleres en una sola slide
  • 🖼️ Fotos carpeta — seleccioná una carpeta local con imágenes o flyers; se rotan al azar
  • ▶ Previsualizar — vista previa simulando la pantalla TV antes de proyectar

💡 Las fotos de la carpeta se cargan solo en esa sesión. Si cerrás el navegador, tenés que volver a seleccionar la carpeta la próxima vez.

🔄 Sincronización

El punto de color en la esquina superior derecha indica el estado de conexión: ● verde = conectado, ● rojo = sin conexión. Todos los cambios se guardan en la nube (Firebase) y están disponibles desde cualquier dispositivo con acceso al sistema.
